🌍 AI Travel Planner for Students

**Project Overview**
AI Travel Planner for Students is a Streamlit-based web application that helps students discover travel destinations based on their budget, interests, state preferences, and trip duration.

The application provides personalized travel recommendations along with destination details, ratings, best season to visit, and destination images.

**Features**

- Budget-based destination recommendations
- Travel interest selection (Adventure, Nature, Historical, Beach, etc.)
- State-wise filtering
- Trip duration selection
- Destination images and details
- User-friendly Streamlit interface
- Fast and lightweight deployment

**Technologies Used**

- Python
- Streamlit
- Pandas
- HTML/CSS
- CSV Dataset

**Future Enhancements**

- AI-based itinerary generation
- Hotel recommendations
- Weather integration
- Google Maps integration
- Personalized travel suggestions using Generative AI
